📝 Smart NoteVerse – Java Mini Project

Smart NoteVerse is a desktop-based text editor developed using Java Swing that allows users to create, edit, open, and save text files efficiently. The project focuses on building a simple and user-friendly interface that supports essential text editing operations such as cut, copy, paste, select all, and basic text formatting like font and color customization.

The application uses Java GUI components such as JFrame, JTextArea, JMenuBar, and JFileChooser to provide an interactive user experience. File handling operations are implemented using Java I/O classes, enabling smooth reading and writing of files. The system follows an event-driven approach where user actions trigger specific functions.

This project helps in understanding core Java concepts like GUI development, event handling, and file management. It serves as a beginner-level foundation for developing more advanced desktop applications in Java.
